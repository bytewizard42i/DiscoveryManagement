# Diligence Pack (Templates)

**Purpose:** This folder contains fill-in-the-blank templates used to make AutoDiscovery investor/diligence materials consistent, auditable, and easy to review.

## How to use

1. Copy each template section and fill in bracketed placeholders like **[TODO: ...]**.
2. Prefer linking to primary sources (statutes, rules, case law, contracts) and attach exhibits where possible.
3. Keep one source of truth per topic (e.g., pricing).

## Templates

- `ADMISSIBILITY_MEMO_OUTLINE.md` — legal admissibility / evidence foundation memo outline
- `CAP_TABLE_TEMPLATE.md` — cap table + financing history template
- `CUSTOMER_DISCOVERY_LOG_TEMPLATE.md` — interview log + insights capture
- `PILOT_LOI_TEMPLATE.md` — lightweight pilot/POC letter of intent template (non-binding)
- `PRICING_ONE_PAGER_TEMPLATE.md` — canonical pricing + packaging one-pager

## Notes

- These are **templates**, not legal advice.
- For anything that touches evidence/admissibility, retain counsel and tailor to jurisdiction.
